Postify is a RESTful blog application built using Node.js,Express.js,EJS,and Tailwind CSS.  
It supports full CRUD operations with a clean and responsive UI.

 Features
- Create, view, edit, and delete posts
- RESTful routing
- Tailwind CSS styled UI
- Server-side rendering with EJS

Tech Stack
- Node.js
- Express.js
- EJS
- Tailwind CSS
